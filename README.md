# Customer Shopping Behavior Analysis

## Overview

This project focuses on analyzing customer shopping behavior using **Python, SQL, and Power BI**. The objective was to transform raw retail data into meaningful business insights by performing data cleaning, exploratory data analysis (EDA), SQL-based querying, and interactive dashboard development. The project highlights customer purchasing patterns, revenue trends, product performance, and customer segmentation to support data-driven decision-making.

---

## Dataset

The dataset contains customer shopping information, including:

* Customer Demographics (Age, Gender)
* Product Categories and Items Purchased
* Purchase Amounts
* Discounts Applied
* Subscription Status
* Shipping Preferences
* Customer Review Ratings
* Previous Purchases and Loyalty Indicators

## Project Workflow

### 1. Data Loading & Preprocessing

* Imported the dataset into Python.
* Checked data structure and quality.
* Handled missing values and inconsistencies.
* Prepared the dataset for analysis.

### 2. Exploratory Data Analysis (EDA)

* Analyzed customer demographics.
* Examined purchase behavior and spending patterns.
* Identified top-performing products and categories.
* Evaluated the impact of discounts and subscriptions.
* Visualized key trends using charts and graphs.

### 3. SQL Analysis

Performed business-focused SQL queries, including:

* Revenue analysis by customer demographics.
* Customer segmentation (New, Returning, Loyal).
* Product and category performance analysis.
* Discount effectiveness evaluation.
* Subscription and repeat purchase analysis.
* Customer spending behavior comparisons.

### 4. Power BI Dashboard

Developed an interactive dashboard to visualize:

* Total Revenue
* Customer Distribution
* Product Category Performance
* Purchase Trends
* Customer Segmentation
* Subscription Analysis
* Discount Impact

### 5. Reporting

Compiled key findings, visualizations, and business recommendations into a structured analytics report.

## How to Run the Project

### Clone the Repository

```bash
git clone https://github.com/your-username/customer-shopping-behavior-analysis.git
cd customer-shopping-behavior-analysis
```

### Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### Run the Jupyter Notebook

```bash
jupyter notebook
```

### SQL Analysis

* Import the dataset into MySQL.
* Execute the SQL queries provided in the `sql/` folder.

### Power BI Dashboard

* Open the `.pbix` file in Power BI Desktop.
* Refresh the data source if required.
